# 🤖 Machine Learning Projects

This repository contains multiple Machine Learning projects implemented using Python. Each project demonstrates different ML algorithms and real-world problem-solving techniques.

## 📂 Projects Included

- Algorithm: Linear Regression

---

### 2️⃣ Spam Classification (Naïve Bayes)
- Classifies messages as spam or not spam
- Algorithm: Naïve Bayes

---

### 3️⃣ Car Price Prediction (Artificial Neural Network)
- Predicts car prices using deep learning
- Algorithm: ANN

---

### 4️⃣ Stock Price Prediction (LSTM)
- Predicts stock trends using time series data
- Algorithm: LSTM (Deep Learning)

---

### 5️⃣ Face Detection
- Detects human faces from images
- Technique: OpenCV Haar Cascade
